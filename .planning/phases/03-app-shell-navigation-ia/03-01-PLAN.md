## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser → shell Link | 用户点击壳层 CTA；目标路径必须为代码常量 |
| server RSC → redirect | dual `/sessions/[id]` 服务端跳转 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Spoofing | `sessions/[id]/page.tsx` redirect | mitigate | 仅 `redirect` 至 `/sessions/${id}/workbench` 或 legacy `getDualWorkbenchRedirect` 结果；禁止 query 拼入 path |
| T-03-02 | Tampering | shell CTAs | mitigate | href 硬编码 `/upload?mode=dual|single`；禁止 `javascript:` |
| T-03-03 | Information disclosure | ContextualShellTitle | accept | 仅静态中文标签，无 PII |
| T-03-04 | Denial of service | — | accept | 无新网络面 |
| T-03-05 | Elevation | — | accept | 沿用 layout `safeGetUser` |
| T-03-SC | Tampering | npm/pip/cargo installs | mitigate | 本 plan 无新包；03-02 安装 collapsible 时走 slopcheck + RESEARCH 审计表 |

</threat_model>
